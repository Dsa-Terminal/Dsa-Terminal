Module pl.Map

A Map class.

> Map = require 'pl.Map'
> m = Map{one=1,two=2}
> m:update {three=3,four=4,two=20}
> = m == M{one=1,two=20,three=3,four=4}
true

Dependencies: pl.utils , pl.class , pl.tablex , pl.pretty
