=====================================================

criamos, quem te revendeu que fez o mal uso! Se mesmo depois de falar com o fornecedor

você recebera seu dinheiro devolta!

Atualizando e redistribuindo!
=====================================================
Você pode fazer alterações no Terminal, mais a não ser que tenha permissão de
Felipe Souza Pereira de Lima para alterar este código isto é uma redisttribuição que você pode
tanto deixa-la de códido-livre tanto cobrar por ela! Mais você perdera o direito de consumidor
deste "SOFTWARE" é qualquer bug que o programa apresentar não e conosco que deve reclamar
você alterou o código-fonte o que pode causar danos au Dsa Terminal! Se você é um usuario que
seu Dsa Terminal falhou constantemente va em https://github.com/Dsa-Terminal/Dsa-Terminal/
crie um poblema que nós o aconselharemos a fazer algo dependendo de que aconteceu, ou pode só ser
o erro do momento que na proxima atualização de "SOFTWARE" ela sera resolvida, mais isso
é só para os usuarios do Dsa Terminal se você o redistribuiu, você não tem o direito de reembolso
solução de poblemas, etc!

Versionando como contribuidor
=====================================================
Se você é um novo contribuidor está parte é para você que tem os direitos de alteração,
sugestão, poder de alterar o código-fonte e continuar tendo o direito de consumidor, etc
